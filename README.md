# SS Brake Pedal: Firmware

> **English:** flashing instructions and the full manual are in [USER_GUIDE_EN.md](USER_GUIDE_EN.md).

Firmware oficial para el **Pedal de Freno de Sanchez SimRacing**.

Aqui se publica cada version del firmware. La actualizacion se hace desde la propia interfaz web del pedal: no necesitas cables, programas ni conocimientos tecnicos.

---

## 1. Descargar el Firmware

1. Abre la seccion **[Releases](https://github.com/sanchezsimracing/ss-brake-pedal-firmware/releases/latest)**.
2. Descarga el archivo **.bin** de la version mas reciente (por ejemplo, `ss-brake-pedal-v1.3.0.bin`).

Guarda el archivo en el mismo dispositivo (telefono o PC) desde el que vas a abrir la interfaz web del pedal.

---

## 2. Conectarte al Pedal

1. Presiona el **boton** del pedal una vez para activar el Wi-Fi.
2. En tu telefono o PC, conectate a la red **SS Brake Pedal** usando tu contraseña.
3. La pagina de configuracion se abre automaticamente. Si no aparece, abre un navegador y ve a **http://192.168.4.1**.

> Si olvidaste la contraseña, manten presionado el boton durante **3 segundos** para entrar en modo de recuperacion. La red se abre sin contraseña y la pagina te mostrara una nueva.

---

## 3. Instalar

1. Desplazate hasta la seccion **Firmware**, al final de la pagina.
2. Toca **Seleccionar archivo .bin** y elige el archivo que descargaste.
3. Toca **Actualizar Firmware** y confirma cuando se te solicite.
4. Una barra de progreso mostrara el estado de la subida. Al terminar, el pedal se reinicia automaticamente con el nuevo firmware.

> **No desconectes el pedal ni cierres la pagina durante la actualizacion.** Espera a que el dispositivo se reinicie por su cuenta.

---

## 4. Verificar

La seccion **Firmware** de la interfaz web muestra la version instalada. Vuelve a abrir la pagina y confirma que coincide con la que descargaste.

---

## Firmware Firmado

Solo se aceptan archivos firmados por Sanchez SimRacing. Si el archivo esta dañado, incompleto o no proviene de este repositorio, la actualizacion se rechaza y el pedal sigue funcionando con su firmware actual.

Descarga el firmware unicamente desde la seccion **Releases** de este repositorio.
